关于 FNS 账号

FNS 账号是适用于联邦知识分享（Federation kNowledge Sharing, FNS）领域的个人账号，它基于 NBC 账号库（NBC Account Library，NAL）构建，是一套支持 DID 自主身份的账号系统。

NAL 软件尚未缓存

NAL 软件需缓存到本地后，才能提供基于 NBC 账号的相关服务，这些服务将以离线方式提供。

在本机创建账号

手机号与助记语一起用于推导账号，其中手机号末尾 4 个数字用作缺省的保留字（以后可改）。助词语（即，脑词）是您最难忘，别人又难猜的一句话，请妥善保管助记语，它属于关键机密，恢复账号会用到。

关于真身验证

真身验证（Real Person Verification）是第三方网站提供真人验证服务，用于确保被服务对象是真实人类。NBC DID 要求真身验证至少要采用手机短信的验证方式，验证通过后第三方网站在它布署的真身服务点（Real Server Point），为用户提供诸如护照颁发、护照验真等服务。

您应该在下面 “设置真身服务点” 指定服务点 URL 并完成真身验证，以便获得可信护照服务，登录应用网站需要可信护照。否则，你只能自行颁发自签护照，自签护照不是可信护照，只适用于少数网站。

真身服务点完成真身验证后，将记录您的手机号与可公开真身账号，然后生成关联凭证。

关于自签护照

自签护照（self-signed passport）是用户自行签名的护照，因为不需第三方机构签名，它可以随时生成、随意更换，隐私性更好，但可信性欠佳。

NAL 库始终配置一份自签护照，应用网站常用它表达当前用户的游客身份，您还可以指定昵称与头像。
